# 0-5岁宝宝生长发育报告

本文介绍基于Python3.5.2，根据世界卫生组织发布的0-5岁宝宝生长发育标准百分位数曲线数据，以及宝宝的身长\高，体重，头围监测数据，
绘制宝宝的身长\高，体重，头围，体重指数，身长\身高别体重的曲线图，然后生成.docx格式的文档，最后将文档转变为PDF格式的报告。
